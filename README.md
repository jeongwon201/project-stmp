# project-stmp | 축구 전술 관리 프로그램

사용자가 응원하는 축구 클럽의 선수 또는 현재 우수한 성적을 거두고 있는 선수를 등록하여 관리하고,  
등록된 선수들을 조합하여 포메이션의 각 포지션에 할당해 나만의 전술을 저장하고 관리할 수 있는 프로그램입니다.  
<br />
<br />

## 프로젝트 배경 및 과정

축구를 좋아하는 사람이라면, 애정하는 클럽이 있기 마련입니다.  

응원하는 클럽이 성적이 저조하거나 아쉽다면,  
"이렇게 포메이션 또는 전술을 구성하면 어떨까 ?" 라는 생각을 종종하곤 합니다.  

또 축구 블로그를 운영 중이거나 커뮤니티 사이트에서 활발히 활동하는 사람이라면,  
위 처럼 생각뿐 아니라 실제 글로 포스팅할 수 있습니다.  

이러한 상황에서 내가 직접 선수와 전술을 구상하고 관리할 플랫폼이 있다면,  
일반 스포츠인에게는 재미, 블로거나 커뮤니티 활동에 있어서 편리할 것이라 생각합니다.  

저는 평소에 축구에 관심이 많아 해외 축구를 자주 보고, 또 즐겨하는 스포츠인입니다.  
축구에 관련된 프로젝트를 하고 싶은 마음이 있었지만 전공 지식이 미흡하여 실천하지 못하였습니다.  

그러나 학업 중 이 프로젝트 기회가 생겨 기획하게 되었습니다.  
<br />
<br />

### 전술 관리
전술 관리 화면입니다.  
<div>
  <img src="readme/images/tm.png" />
</div>
<br />

화면에서 보이는 532 포메이션 외 여러 포메이션을 지원하며,
